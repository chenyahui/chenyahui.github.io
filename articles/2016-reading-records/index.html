2016 年读了哪些书？

发表于 2016-12-31 更新于 2019-12-10 分类于 生活和思考 阅读次数： Valine：

2016 年总共阅读了 25 本书, 其中计算机相关书籍 5 本，小说和传记类 15 本，其他类型的 5 本。阅读介质大部分为 kindle 电子书和微信读书 APP。
2016 年时间匆匆，感谢我看过的每一本书。

计算机类书籍

书名 推荐程度 日期 评价
Java 并发编程实战 四星 2016 Java 程序员必读。这本书给四星。五星给原著，一星给翻译。
Redis 设计与实现 五星 2016 国人写的好书。惭愧的是，我看此书之前基本没怎么用过 redis。但是看了此书，让我不再惧怕新技术
Go 语言编程 五星 2016 为了做 MIT 6.824 看的此书。对 Go 语言的总体评价是，它是一门极度简洁、高效而又奇怪的工程化语言
Effective Java 四星 2016-12-26 Java 必读。精华很多，但同时废话也很多。
深入理解 Java 虚拟机 五星 2016-12-27 也是国人写的好书。今年读的最好的技术书。GC 算法和多线程两部分讲的非常好，受益匪浅。

消遣类书籍

书名 推荐程度 日期 评价
淘宝技术这十年 五星 2016-04-07 给我的震撼不次于当初看电影《社交网络》的时候
暗时间 三星 2016-04-18 算是本好书，但是够不上它的名声
Mactalk 跨越边界 三星 2016-04-18 微信集？？
激荡三十年 五星 2016-10-31 感触最深的一本书。我经常在想，假如我活在那个年代，怎么才能和他们做的一样好。让我印象最深的是海尔的张瑞敏的座右铭，“战战兢兢，如履薄冰”。时代和命运的造化，个人的努力，才是成人成事的关键。
邓小平改变中国 四星 2016-11-19 毛时代是如何过渡到邓时代的

小说 / 传记类

书名 推荐程度 日期 评价
没有色彩的多崎作和他的巡礼之年 五星 2016-05-10 一个寻找自我的过程
白夜行 五星 2016 从此成为东野圭吾粉
嫌疑人 X 的献身 五星 2016
汪曾祺小说选 五星 2016 很遗憾这么晚才读汪曾祺先生的书。汪曾祺先生的作品是遗世独立而又烟火十足的清流。
未来边缘 四星 2016 大刘的书
蛙 五星 2016-8-27 计划生育苦难史。虽然莫言的作品以荒诞著称，但是这本书里面的历史确实真实存在在离我们不远的过去。
毛泽东传 五星 2016-9-3 红太阳是如何升起的。听朋友说国内看到的都是删减版的，希望有机会可以看到完整版。
一句话顶一万句 五星 2016-9-22 刘震云的语言魅力。虽然语言整体很轻松，但是作品从头到尾都流露出一种无人可说的孤独感。此外，刘震云和我是老乡，这部作品里把很多地道的河南话写到纸面上，奇怪而又亲切。
有味 四星 2016-10-13
恶意 五星 2016-10-31
秘密 五星 2016-11-11
解忧杂货店 五星 2016
一个陌生女人的来信 五星 2016-11-26 情不知所起，一往而深
绝代双骄 五星 2016-12-23
偷影子的人 五星 2016-12-23

25 本书的数量实在不算多，但是比着过去两年要进步很多。非常感谢 Amazon 的 kindle 给了非常好的阅读体验。腾讯的微信读书 APP 也是一个我个人觉得目前市面上最好的阅读软件，其中 18 本书都是在它上面读的。kindle 和微信读书让我利用了很多碎片时间，比如等人，等车，地铁等的时间，一年下来也非常惊叹碎片时间的可利用性如此强大。
总的来说，今年在技术书籍方面阅读量相对较少，严肃类文学书籍较少，在接下来的一年将会有所调整。
