2015 年度总结

发表于 2016-01-01 更新于 2019-12-17 分类于 生活和思考 阅读次数： Valine：

2015 年都做了什么？

学习

很忙碌的一年。
**2015 年 1 月份到 2 月份 **
基本是在忙数学建模的美赛，最后碰运气拿到了二等奖，加上 2014 年拿到的国赛二等奖。推免的资格基本没什么问题了。
大三下学期
踏踏实实的做了回学霸，认真上课，做实验和课设。在此期间参加了腾讯及阿里巴巴的暑期实习生校园招聘，两次一面跪，备受打击，同时发现了自己在技术上的诸多不足。

读《算法导论》，之前怎么也听不懂的数据结构突然感觉明白了好多。
看《C++ Primer》《STL 源码剖析》, 学 C++。这次对 C++ 有了新的认识，虽然目前 C++ 还是写的很烂，但在接下来的时间里依然会 不断的练习
接触数据挖掘，感觉挺有意思的，基于贝叶斯写了一个分类器，匹配率大约在 85%。在暑期拓展训练的题目里，选了一个爬虫和搜索的题目，基于 PHP 和 sphinx 做的。
以上两个算是小程序，但对我的启发很大。这两个程序的共同点是需要处理的数据量都非常大，用之前写简单程序时的思维处理的时候显然不行，我的分类器刚开始的时候非常慢，慢到一个小时可能就处理一点点。后来做了简单的优化，算是基本能用。 通过这个，让我更加意识到，程序处理大数据量时的思路以及优化的重要性，仅仅 Demo 式的写程序是不可行的。
保研
暑假期间海投各个学校的夏令营，但因为大一大二的时候绩点太低，只有夏令营武大、华东师范大学的过了初审。
武汉大学的夏令营还是有点坑的，考数学，除了考试头天晚上看了下数学，就没什么准备，自然就跪了。 武大和华东师范大学有点冲突，最后也就没去。
9 月份推免的时候得到了武大、华科、北航、中科院信工所的面试机会。
其中也有波折，非常想去的北航把复试定到了 10 月 10 号，而武大的 2 号面试，当天要求回复。我选择了最稳妥的结果，读武大计算机学院的专硕。
虽然心有不甘，但是有时想想人生就是这个样子，并没有真正的一帆风顺和万事如意。
保研之路并不顺利，但是为保研努力的过程中却收获了不少。
暑假这段时间为准备夏令营，把专业课特别认真的复习了一遍，看了很多专业书籍，刷了 200 多道 Leetcode，自己的基本功扎实了不少。看 APUE 和 UNP 的时候对 Linux 产生了浓厚的兴趣，之后基本把操作系统换成了 Ubuntu。
大四上学期
大四上学期感觉时间很短，基本从 10 月初保研落下帷幕，离现在也就 2 个月的时间了。在这两个月期间也做了很多事情。
先是进了武大的实验室，憧憬着做感兴趣的大数据，万万没想到第一个项目竟然是最讨厌的 Java Web。开始学 Java、SSH 三个框架，慢慢的体会到实际上 Java 这语言并没有那么烂，它的成功有很多可取之处。我从这个项目里面也学到了一些东西。
然后接了同学介绍的外包项目，从头到尾的做整个网站，赚了些外快，同时使我的前端方面进步了很多。

项目

  1. 完成了 “理工发布” 后台和接口的开发，从 PHP 的入门小白到第一次实战。虽然写的代码各种烂，但是学到了很多的东西。
  2. 完成了 “助学方舟”，虽然都是用 DeDecms，但是使我了解到了一个完备的 CMS 应有的功能。
  3. 完成了长江水利委一个 OA 系统。第一次写 Java 程序，第一次写 Java Web, 第一次写 SSH…
  4. 外包网站项目。一个人从设计、前端、后台搞定，还是学到了很多东西，尤其是前端方面的。
  5. 接触某爬虫项目。虽然质量很水，还是有很多东西值得学习。在接下来的时间里会把它完善到更好。
  6. 各种小程序…

总结

非常忙碌的一年，也是丰收的一年。这一年让我很清楚地认识到自己只不过是个凡人，并没有什么毫无根据的天生骄傲。踏实的学习，认真的思考，每个行业都是这样，每件事情都应该是这样。
这一年做的事情比前两年都要多，但是感受到的确实认真学习时充实的快乐。也没想到的是，大三认真的上课让我拿到了国家励志奖学金，这是我大学第一份奖学金。
有时候我也会想，如果我大一大二不去搞社团，忙活动，而是专心学习，结果会不会好点。仔细思考，我觉得我并没有什么可后悔的，大一大二的社团活动，给了我非常珍贵的经历，让我整个人的心情变得更自信、开朗和更加坚韧，也收获到了很多朋友吧，这是无论如何也都无法替代了。有的只能是遗憾，但绝不会后悔。
